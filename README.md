# JMZS – Lerntools

Kleine Sammlung von Lernhilfen für die Schule.

## Seiten

### [Lernwörter](lernwoerter.html)
Zufallsgenerator für Lernwörter. Drücke den Knopf, um ein zufälliges Wort aus der Liste anzeigen zu lassen. Jedes Wort erscheint nur einmal – die Liste wird kleiner, bis alle Wörter geübt wurden. Ein Fortschrittsbalken zeigt, wie viele Wörter bereits dran waren. Mit „Neu starten" beginnt man wieder von vorne.
